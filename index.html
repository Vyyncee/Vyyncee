Définitions

La page web est obtenue par interprétation d'un code HTML envoyer par un serveur en réponse de la requete qui lui a etait envoyée (adresse en http://www.etc). C'est le navigateur qui interprete ce code.

Il existe deux sortes de balises :

  1. Les balises les plus courantes encadrent du code qui se termine par une balise dite fermante. Exemple : <balise> Contenue HTML </balise>
  2. Les balises orphelines qui n'encadrent aucun code et qui ce ferment toutes seules. Exemple : <br/>, <img/>, <input/> etc...
